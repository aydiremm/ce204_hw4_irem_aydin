void ce204_hw4_app.Menus.VendorMenu.run ( )

Executes the main functionality of the class.

This method represents the main entry point of the class and is responsible for executing its core functionality. It displays a menu for vendors, allowing them to add or remove cars, or exit the menu. The user's choice is read from the input using a Scanner object. Based on the chosen option, the corresponding method is called to perform the requested action. The loop continues until the user chooses to exit the menu.
