\doxysubsection{Detailed Description}
Represents the main menu of the Car Rental Management System. 

\begin{DoxyVerb}   The MainMenu class extends the Menu class and provides functionality
   for displaying the main menu options and handling user input. It
   allows users to select their role (vendor or user) or choose to exit
   the program.
\end{DoxyVerb}
